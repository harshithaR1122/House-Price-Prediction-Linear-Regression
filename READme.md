# House Price Prediction using Linear Regression

## Internship
MainCrafts AI & ML Internship

## Task 1: Build & Evaluate a Linear Regression Model

### Project Overview
This project focuses on predicting house prices using a Machine Learning model. A Linear Regression algorithm is implemented to understand the relationship between various housing features and the median house value.

### Dataset
The project uses the California Housing Dataset containing information such as:

- Longitude
- Latitude
- Housing Median Age
- Total Rooms
- Total Bedrooms
- Population
- Households
- Median Income
- Ocean Proximity

### Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

### Project Workflow

1. Data Loading
2. Exploratory Data Analysis
3. Handling Missing Values
4. Encoding Categorical Features
5. Splitting Data into Training and Testing Sets
6. Training Linear Regression Model
7. Model Evaluation
8. Saving Trained Model

### Model Evaluation

The model was evaluated using:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score

### Results

The Linear Regression model achieved:

- MAE: 50670.73
- MSE: 4908476721.15
- R² Score: 0.6254

### Files Included

- `task1_ml_linear_regression_final.ipynb` - Complete implementation notebook
- `housing.csv` - Dataset used for training
- `linear_regression_house_price_model.pkl` - Saved trained model

### Conclusion

This project demonstrates the complete machine learning workflow for house price prediction, including data preprocessing, model training, evaluation, and model saving.
